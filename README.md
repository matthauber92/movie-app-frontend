# React Frontend

This project is a **React App with TypeScript** and the following stack:

- [**Ant Design**](https://ant.design/): UI Library
- [**Redux**](https://react-redux.js.org/): Global application state management tool
- [**Husky**](https://typicode.github.io/husky/): Git linting tool
- [**Vite**](https://vitejs.dev/guide/): Frontend build tool

## Getting started

### 1. Install dependencies

```sh
npm install
```

### 2. Start Application

Run the following command to start your react application.

```sh
npm run dev
```

## TypeScript Generation

Run the following command to generate typescript types

```sh
npm run generate
```
